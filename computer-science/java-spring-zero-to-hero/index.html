Diventa esperto
di Java, Spring e Spring Batch

Un percorso completo in italiano: dai fondamenti del linguaggio Java alla JVM interna, da Spring Core a Spring Boot, da Spring Data a Spring Security, fino alla padronanza di Spring Batch in ogni suo dettaglio. Teoria universitaria resa masticabile, centinaia di esercizi con soluzione, diagrammi SVG e Mermaid, esempi production-grade.

46 sezioni Esercizi guidati Spring Batch deep Bilingue IT / EN

Come si usa

Le sezioni sono in ordine logico: ogni capitolo aggiunge mattoni sul precedente. Non saltare i fondamenti anche se sai già qualcosa: in Java l'80% dei bug nasce dal non capire bene la JVM, la concorrenza o il modello transazionale di Spring. Ogni pagina ha quattro parti:

  • Teoria — il come e il perché. Niente magia: capire i meccanismi.
  • Esempi di codice — runnabili, commentati, con output atteso.
  • Diagrammi — flussi, gerarchie, sequenze. Tutto in SVG o Mermaid, mai ASCII art.
  • Esercizi — con soluzione nascosta, dal facile al "rompi il sistema e capisci perché si rompe".

Cosa ti serve per iniziare

  • JDK 21 (LTS corrente). Scarica Temurin, Liberica o Oracle JDK.
  • Un IDE serio: IntelliJ IDEA Community (gratis, copre il 95% di ciò che serve) o VS Code con Extension Pack for Java.
  • Maven e/o Gradle installati nel PATH.
  • Docker Desktop per girare Postgres/Kafka/Redis in locale, soprattutto per la parte Spring Boot e Batch.
  • Un account su start.spring.io non serve, ma è il bookmark più importante.
  • Curiosità e perseveranza: questo non è un percorso breve.
