Become an expert
in Java, Spring and Spring Batch

A complete path in English: from Java language fundamentals to JVM internals, from Spring Core to Spring Boot, from Spring Data to Spring Security, all the way to total mastery of Spring Batch. University-level theory made chewable, hundreds of exercises with solutions, SVG and Mermaid diagrams, production-grade examples.

How to use

Sections follow a logical order: each chapter builds on the previous one. Don't skip the fundamentals even if you already know something: in Java, 80% of bugs come from not truly understanding the JVM, concurrency, or Spring's transactional model. Each page has four parts:

  • Theory — the how and why. No magic: understand the mechanisms.
  • Code examples — runnable, commented, with expected output.
  • Diagrams — flows, hierarchies, sequences. All in SVG or Mermaid, never ASCII art.
  • Exercises — with hidden solutions, from easy to "break the system and understand why it breaks".

What you need to start

  • JDK 21 (current LTS). Get Temurin, Liberica or Oracle JDK.
  • A serious IDE: IntelliJ IDEA Community (free, covers 95% of needs) or VS Code with the Extension Pack for Java.
  • Maven and/or Gradle in your PATH.
  • Docker Desktop to run Postgres/Kafka/Redis locally, especially for the Spring Boot and Batch sections.
  • A bookmark on start.spring.io.
  • Curiosity and perseverance: this is not a short path.
