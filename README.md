## How to Use

### 🌐 Method 1: Online (Recommended)
Simply click on any of the links above. The courses are hosted via **GitHub Pages**, allowing you to read, study, and interact with the material immediately on your PC, tablet, or smartphone without downloading anything.

---

### 💾 Method 2: Offline / Local Execution
If you prefer to have a local copy on your machine:
1. **Download** the specific subject folder (or clone the entire repository).
2. Locate the `index.html` file inside the course folder.
3. **Double-click** `index.html` to open it locally inside any modern web browser (Chrome, Firefox, Safari, Edge).
4. Enjoy your study session!
